# 🎨 Sentinel Skins

**Rung 0 · the Beauty axis.** Makes any NinjaTrader 8 chart — and the whole platform — look
considered. Zero intelligence required.

## What's here
- **7 themes** — Dark · Light · Silver · Obsidian · Blueprint · Amber · Neon (`themes/`)
- **SentinelWallpaper** — a theme-aware ghosted-helmet chart wallpaper (`Indicators/`)
- Draws with **SentinelSkin**, the suite's glass-card framework (in [`../runtime/`](../runtime/))

## Install
1. Copy [`../runtime/`](../runtime/) (the shared runtime — once) and this folder's `Indicators/`
   into `Documents\NinjaTrader 8\bin\Custom\`.
2. Copy the `themes/Sentinel*` folders into `Documents\NinjaTrader 8\templates\Skins\`.
3. Press **F5** in the NinjaScript Editor.
4. Apply a theme via **Tools → Options → General → Skin**.

> Cyan means *live and watching* — it's the only accent. Green and red are money and direction.
