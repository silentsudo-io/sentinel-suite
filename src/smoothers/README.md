# 〰️ Sentinel Smoothers

**Rung 1 · Beauty × Data.** Every moving average and filter worth having, clean-room and rebuilt to
the suite's card and color language.

## What's here (23)
- **Classics** — SMA · EMA · WMA · VWMA · DEMA · TEMA · HMA · TMA · TWMA · TSMA · DWMA · DTMA · DSMA
- **Zero-lag family** — ZLEMA · Zero-Lag TEMA · Zero-Lag HA-TEMA · HoltEMA
- **Regression / stat** — LinReg · Moving Median
- **DSP filters (Ehlers)** — Super Smoother · Gaussian · Butterworth · Non-Linear (Ehlers)

Each implements a public / textbook formula; the code is original to this project.

## Install
1. Copy [`../runtime/`](../runtime/) (shared runtime — once) and this folder's `Indicators/` into
   `Documents\NinjaTrader 8\bin\Custom\`.
2. Press **F5** in the NinjaScript Editor.
3. Right-click a chart → **Indicators** → **Sentinel**.
